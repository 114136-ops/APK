## 繁體中文

### 🌟 簡介
這是一款專為忙碌且偶爾會忘東忘西的媽媽們量身打造的冰箱管理工具。透過直覺的介面與智慧化的提醒，我們幫助您輕鬆掌握冰箱內的每一份食材，減少浪費，讓下廚變得更輕鬆愉快。

### ✨ 核心功能
*   **智慧食材管理**：記錄食材名稱、數量、分類（青菜、肉類、飲料等）及位置（冷藏/冷凍）。
*   **愛的對話提醒**：系統會根據食材種類發出溫馨的對話式通知（例如：「媽媽～牛奶快到期了，快把它喝掉吧！🥛」）。
*   **買菜清單**：隨手記下需要補充的物品，支援一鍵勾選。
*   **一鍵分享**：可將買菜清單或到期食材整理成文字，快速分享至 Line 或簡訊給家人。
*   **快速搜尋與過濾**：不管是想找「蛋」還是想看「冷凍庫」裡有什麼，按一下就能找到。
*   **可愛溫馨介面**：使用圓潤的圓體字與現代化的 Material 3 設計。

### 🛠️ 技術架構
*   **開發語言**：Kotlin
*   **介面框架**：Jetpack Compose (Material 3)
*   **資料儲存**：Room Database (支援資料永久保存)
*   **背景任務**：WorkManager (自動執行每日到期檢查)
*   **設計風格**：MVVM 架構

## English

### 🌟 Introduction
This is a fridge management assistant tailor-made for busy moms who might occasionally forget what's in their kitchen. With an intuitive interface and intelligent reminders, we help you keep track of every ingredient, reduce food waste, and make cooking a more joyful experience.

### ✨ Key Features
*   **Smart Ingredient Management**: Track item names, quantities, categories (Veggies, Meat, Drinks, etc.), and storage locations (Fridge/Freezer).
*   **Friendly Chatty Notifications**: Receive warm, conversational reminders based on food types (e.g., "Mom~ the milk is expiring soon, let's drink it! 🥛").
*   **Shopping List**: A handy place to jot down items you need to restock, with simple check-off functionality.
*   **One-Click Sharing**: Format your shopping list or expiring items into a text message and share it with family members via Line or SMS.
*   **Quick Search & Filtering**: Instantly find what you're looking for by name or category.
*   **Warm & Modern UI**: Featuring rounded "cute" typography and a sleek Material 3 design.

### 🛠️ Technical Stack
*   **Language**: Kotlin
*   **UI Framework**: Jetpack Compose (Material 3)
*   **Persistence**: Room Database (Ensures data is never lost on restart)
*   **Background Tasks**: WorkManager (Automated daily expiry checks)
*   **Architecture**: MVVM (Model-View-ViewModel)
